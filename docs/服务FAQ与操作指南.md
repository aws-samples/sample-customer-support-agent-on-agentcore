# 服务FAQ与操作指南

## 约课操作

### 如何约课
约课步骤：
1. 打开 APP，进入首页
2. 点击「约课」按钮
3. 选择想要的老师和可用时间段
4. 确认课程类型和学生信息
5. 点击「确认预约」完成约课

约课注意事项：
- 约课需要在开课前 **24小时** 完成
- 每次约课需消耗对应课程类型的课时余额
- 可以选择指定老师，也可以由系统推荐

### 如何取消课程
取消课程步骤：
1. 打开 APP，进入「我的课程」
2. 找到需要取消的课程
3. 点击课程卡片上的「取消」按钮
4. 确认取消操作

取消规则：
- 开课前 **12小时以上** 取消：全额退还课时，不收取费用
- 开课前 **12小时以内** 取消：将扣除对应课时
- 缺课（未取消且未上课）：同样扣除课时

### 如何调课（改时间）
调课步骤：
1. 打开 APP，进入「我的课程」
2. 找到需要调整的课程
3. 点击「调课」按钮
4. 选择新的时间和老师（可选更换老师）
5. 确认调课

调课规则：
- 调课需在开课前 **12小时** 操作
- 可以同时更换老师和时间
- 调课不额外收取费用

## 请假规则

请假操作路径：APP → 我的课程 → 选择课程 → 请假

请假规则：
- 需要在课前 **12小时** 操作
- 12小时内请假视同取消，将扣除相应课时
- 未请假且缺课将扣除课时
- 请假后课时将退还到账户余额

## 打卡活动

### 如何参与打卡
1. 进入 APP「活动中心」
2. 点击当前进行中的「打卡活动」
3. 按照活动要求完成打卡任务（如：分享学习内容到朋友圈）
4. 生成专属打卡海报
5. 将海报分享到朋友圈
6. 截图上传到 APP 等待审核

打卡须知：
- 每周最多参与一次打卡
- 审核通过后可获得积分奖励
- 积分可在积分商城兑换礼品或课时

## 积分规则

### 积分获取方式
- 打卡活动：每次打卡成功获得积分
- 推荐好友：成功推荐新用户注册并购课
- 课程评价：完成课后评价

### 积分使用规则
积分可用于：
1. 兑换额外课时
2. 兑换学习礼品（文具、教材等）
3. 抵扣部分课程费用

积分有效期：自获得之日起 **12个月** 内有效，请及时使用。

## 设备检测问题

### 麦克风/摄像头异常
如果遇到设备检测问题，请按以下步骤排查：
1. **检查设备权限**：确保 APP 已获得麦克风和摄像头权限
2. **重启 APP**：关闭 APP 后重新打开
3. **检查网络连接**：确保网络稳定
4. **检查设备本身**：尝试在其他应用中使用麦克风/摄像头
5. **重新安装 APP**：如以上方法无效，尝试卸载后重装

### 上课卡顿或黑屏
- 检查网络带宽是否满足要求（建议 5Mbps 以上）
- 关闭其他占用网络的应用
- 尝试切换 WiFi 或移动网络
- 如持续出现问题请联系客服

## 常见问题

### 课时余额怎么查询？
打开 APP → 我的 → 账户信息，可以查看各课程类型的剩余课时和到期时间。

### 如何更换老师？
约课或调课时都可以选择不同的老师。如果想查看某位老师的可用时段，可以在约课页面筛选指定老师。

### 课程可以退款吗？
课程退款属于敏感操作，请联系您的专属顾问处理。客服助手无法直接操作退款。
